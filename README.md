# Storybook Report Archive

Auto-generated Storybook component library build archive for [xabierlameiro/the-last-dance](https://github.com/xabierlameiro/the-last-dance), deployed to GitHub Pages.

## What’s in this repo

The `storybook-static/` folder contains the latest static Storybook build. It is updated automatically on every CI run.

## Viewing the Storybook

Open the [GitHub Pages site](https://xabierlameiro.github.io/stb-report/) to browse the component library.

## Source

Components are defined in [xabierlameiro/the-last-dance](https://github.com/xabierlameiro/the-last-dance) under `src/components/`.
