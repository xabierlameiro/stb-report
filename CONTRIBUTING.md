# Contributing

This repository contains auto-generated Storybook component library builds. It is updated automatically by CI workflows from the source repository.

## How this works

1. CI workflow in [the-last-dance](https://github.com/xabierlameiro/the-last-dance) runs `build-storybook`
2. The static build is committed to `storybook-static/` in this repository
3. GitHub Pages serves the Storybook at the published URL

If you find a bug, please open an issue in the [source repository](https://github.com/xabierlameiro/the-last-dance).
